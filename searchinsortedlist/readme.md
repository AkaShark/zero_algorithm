# 在排序数组中查找数字 I

##### 统计一个数字在排序数组中出现的次数。

**示例1**
> 输入:   
> nums = [5,7,7,8,8,10], target = 8   
> 输出:    
> 2

**示例2**
> 输入:   
> nums = [5,7,7,8,8,10], target = 6   
> 输出:   
> 0   

**提示**
+ 0 <= nums.length <= 105
+ -109 <= nums[i] <= 109
+ nums 是一个非递减数组
+ -109 <= target <= 109

**思路**
> 思路1   
> 双指针   
> 左指针从左到右找到第一个等于target的元素下标，右指针从由到左找到第一个等于target的元素下标
> 两者的距离+1即为结果。
>
> 思路2   
> 二分法   
> 使用两次二分法，分别找到值为target的元素序列的边界，然后计算个数。
